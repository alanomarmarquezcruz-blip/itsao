Descripción de la Asignatura

La asignatura de Desarrollo Full Stack prepara a los estudiantes en el diseño, desarrollo e implementación de aplicaciones web completas. Incluye el desarrollo del lado del cliente (Front-End) y del servidor (Back-End), utilizando tecnologías modernas para crear sistemas web accesibles, responsivos y escalables.

Competencias a Desarrollar

  • Diseñar interfaces web accesibles y responsivas.
  • Desarrollar aplicaciones web completas.
  • Implementar arquitecturas cliente-servidor.
  • Aplicar patrones de diseño en el desarrollo Back-End.
  • Usar frameworks modernos como Angular.
  • Aplicar control de versiones y despliegue continuo.

Temario Principal

1. Fundamentos del Desarrollo Full Stack

  • Diferencias entre Front-End, Back-End y Full Stack.
  • Arquitectura cliente-servidor.
  • Navegadores y servidores web.
  • Control de versiones con Git y GitHub.

2. Interfaces Web Modernas

  • HTML5 y estructura semántica.
  • CSS3 y diseño responsivo.
  • Frameworks Bootstrap y Tailwind.
  • Accesibilidad web.

3. Arquitectura Back-End

  • Patrones de diseño como Singleton y Factory.
  • Patrones estructurales y de comportamiento.
  • Desarrollo de sistemas escalables.

4. Desarrollo Front-End con Frameworks

  • Introducción a Angular.
  • Consumo de servicios REST.
  • Uso de Firebase.
  • Formularios reactivos y validaciones.

Herramientas Utilizadas

  • Visual Studio Code
  • Node.js y NPM
  • Git y GitHub
  • Angular
  • Firebase
  • Bootstrap y Tailwind CSS

Proyecto de la Asignatura

El proyecto consiste en desarrollar una aplicación web funcional donde se apliquen los conocimientos adquiridos. Incluye las siguientes fases:

  • Fundamentación: Investigación y análisis del problema.
  • Planeación: Diseño del sistema y planificación.
  • Ejecución: Desarrollo e implementación.
  • Evaluación: Revisión de resultados y mejoras.
